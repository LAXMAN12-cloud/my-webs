A web page is a document or resource of information, i.e., created for the World Wide Web and can be accessed through a web browser. You can access a web page either by clicking a hyperlink or entering a URL in the browser. The server, where the page is stored returns the HTML document to your computer and the browser displays it. You may interact with this displayed document through various available links. There are two categories of web pages: Static and Dynamic web pages. As.the name implies, static means fixed or stationary, therefore, the static web pages returns information, i.e., predefined and pre-formatted at the time of page creation. For example, the present Home page ofIGNOU web site. On the other hand on adynamic Web page, you can make requests for data, i.e., available in a database on the server and the required information is assembled on the fly. For example, to book a ticket you want to find outznformation about ticket availability for particular dates, ticket fare etc. When you select these options, the request is relayed to the server, the server compiles the relevant information specifically for your request and sends it back to your browser.
